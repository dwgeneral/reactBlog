sublime 小记

最近尝试了一下 sublime Text 编辑器，被它的强大功能吸引了，而且它和 Vim 的操作习惯还算统一，所以可以玩玩！ 1. 多点编辑： 按住 Command ＋ 鼠标点击 2. 呼出左侧 folder : Command + K + B 3. 呼出命令面板 : Command + Shift + p 4. 快速打开一个文件： Command + p (文件名/文件名@关键字）光标自动跳转到关键字位置 5. 快速创建文件： Command + Option + n 持续更新。。。
